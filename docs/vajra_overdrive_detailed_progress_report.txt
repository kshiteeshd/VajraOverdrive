  ----------------------
  4.3 GAME LOOP SYSTEM
  ----------------------

File: core/GameLoop.java

Purpose: Runs the continuous game execution loop.

Loop Characteristics:

Target Update Rate: 60 updates per second

Game Loop Flow:

while(running)

    update()
    render()

Internal Update Flow:

GameLoop ↓ GameCanvas.update() ↓ EntityManager.update()

Render Flow:

GameLoop ↓ GameCanvas.repaint() ↓ paintComponent()

FPS Counter:

Displays frame rate in the top-left corner of the screen.

  4.4 INPUT SYSTEM
  ------------------

File: input/InputManager.java

Purpose: Handles keyboard input using the KeyListener interface.

Key Handling:

KeyPressed → sets key state TRUE KeyReleased → sets key state FALSE

Input is stored in a boolean array allowing any system to check key
state using:

InputManager.isKeyPresent(KeyEvent)
