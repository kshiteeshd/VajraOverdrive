2. PHASE 1 – CORE FOUNDATION (DAY 1–7)
------------------------------------------------------------

Objectives

Establish project infrastructure and core engine systems.

Tasks

Initialize Git repository
Setup project structure
Configure build system (Gradle or Maven)
Create main application launcher
Implement game window
Implement rendering canvas
Implement fixed timestep game loop

Create basic entity class
Create basic sprite rendering pipeline

Deliverable

A running application window with a functioning game loop and rendering system.

------------------------------------------------------------

Objectives

Develop the primary gameplay mechanics.

Tasks

Implement player entity
Implement player movement system

Implement enemy base entity
Implement enemy spawning system
Implement collision detection system
Implement damage system
Implement score system
